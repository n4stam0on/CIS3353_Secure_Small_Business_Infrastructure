# Evidence Directory

This directory documents all collected evidence generated throughout the project, including screenshots, logs, and packet captures used to support analysis and validation.

Due to repository size and file limitations, raw evidence files (such as PCAPs and large logs) are stored externally. This directory provides references and verification details for those files.

---

## Evidence Format

Each piece of evidence is documented using the following format:

- **File Name:**  
- **Description:**  
- **Date Captured:**  
- **Associated User Story / Issue:**  
- **SHA256 Hash:**  
- **Storage Location:** (Google Drive / OneDrive link)

## Notes

- All evidence aligns with specific user stories and milestones within the project  
- Hash values are included to ensure integrity and prevent tampering  
- Screenshots used in documentation are derived from these original evidence files  

---

## Summary

This structure ensures that all evidence is traceable, verifiable, and organized in alignment with the project’s security analysis and validation process.
